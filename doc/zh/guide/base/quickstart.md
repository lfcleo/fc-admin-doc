# 快速开始

## 开发环境

请自行安装`node`和`npm`。

作者开发环境

node版本：`v20.13.1`

npm版本：`v10.5.2`

## 从 GitHub 上拉取

完整版前端代码

```shell
git clone https://github.com/lfcleo/fc-admin.git
```

## 从 Gitee 上拉取

完整版前端代码

```shell
git clone https://gitee.com/lfcleo/fc-admin.git
```

## 安装依赖

使用终端进入项目文件夹中

```shell
npm install
```

## 启动项目

```shell
npm run dev
```

在浏览器中使用 http://localhost:2800 查看运行项目

# 介绍
FC-Admin是一款开源免费且开箱即用的中后台管理系统模版。使用了最新的 Vue3、Vite、Element-Plus、TypeScript、Pinia 等主流技术开发。助您加快开发速度，节省大量时间成本。

## 精简版本

精简版是基于 FC-Admin 提炼出的架子，包含主体功能，更适合实际项目开发。开发者只需要在此项目上做“加法”开发，无需从完整版做“减法”。

GitHub：https://github.com/lfcleo/fc-admin-thin
<br />
Gitee：https://gitee.com/lfcleo/fc-admin-thin

## 灵感来源

本项目基于[SCUI](https://gitee.com/lolicode/scui)开发。由于 SCUI 作者已经好久没维护，所以 FC-Admin 基于 SCUI 的样式，从零搭建代码开发，使用了最新的技术栈，并修复了 SCUI 中的 bug。在这里感谢 SCUI 作者的开源项目。

FC-Admin将会持续更新，修复 bug，增加新功能，欢迎 Star 和 Issues。

# 主题

内置了丰富的主题，并且可根据颜色选择器更改主题色。

![theme](/images/guide/theme.png)

## 设置默认主题色

`/src/store/system/global.ts`文件中的`sysColor`字段是设置默认主题色。

[点击查看代码](https://github.com/lfcleo/fc-admin/blob/main/src/store/system/global.ts#L27)

## 添加快捷主题色列表

`/src/layout/views/setting.vue`文件中的`colorList`字段可修改快捷主题色。

[点击查看代码](https://github.com/lfcleo/fc-admin/blob/main/src/layout/views/setting.vue#L104)

# 暗黑模式

`/src/store/system/global.ts`文件中的`sysDark`字段是设置默认明亮/暗黑模式。

[点击查看代码](https://github.com/lfcleo/fc-admin/blob/main/src/store/system/global.ts#L29)

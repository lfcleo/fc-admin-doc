# 国际化

框架已经集成了国际化支持，只需要在`/src/locale/lang/`目录下设置即可，默认已经添加了中文和英文，你可以根据需要添加其他语言。

## 使用方法

如果需要新增加一些字段使用中文和英文，则只需要编辑`/src/locale/lang/`目录下的`en.ts`和`zh-cn.ts`文件即可。使用方法如下：

1. 在`/src/locale/lang/`目录下的`en.ts`和`zh-cn.ts`文件中添加需要的字段

2. 页面中需要使用的地方，直接使用`$t('key')`即可
```vue
<template>
...
<p>{{ $t('login.describe') }}</p>
...
</template>
```

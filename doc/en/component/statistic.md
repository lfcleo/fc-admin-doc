# 统计数值

[查看预览](https://www.fcadmin.fun/#/vab/data/statistic)

## 使用方法

```vue
<fc-statistic title="标题" value="112893" description="介绍" tips="Tips" groupSeparator></fc-statistic>
```

## 参数说明

### Attributes

属性名  |   说明    |   类型    |   默认值
:----:|----|:----:|:----:|
title | 标题 | string | - |
value | 内容 | string | - |
prefix | 前缀 | string | - |
suffix | 后缀 | string | - |
description | 描述 | string | - |
tips | 提示 | string | - |
groupSeparator | 是否计算属性 | boolean | - |

# Internationalization

The framework has integrated support for internationalization, which can be customized in the `/src/locale/lang/` directory. By default, Chinese and English have already been added, and you can add other languages as needed.

## Usage

To add new fields for use in both Chinese and English, simply edit the `en.ts` and `zh-cn.ts` files located in the `/src/locale/lang/` directory. Here's how to do it:

1. Add the required fields in the `en.ts` and `zh-cn.ts` files under the `/src/locale/lang/ `directory.

2. Wherever you need to use these translations on your pages, directly utilize `$t('key')`.

```vue
<template>
...
<p>{{ $t('login.describe') }}</p>
...
</template>
```

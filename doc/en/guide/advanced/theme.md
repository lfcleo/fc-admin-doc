# Theme

Built in rich themes, and the theme color can be changed according to the color selector.

![theme](/images/guide/theme.png)

## Set default theme color

The `sysColor` field in the `src/store/system/global.ts` file is used to set the default theme color.

[show code](https://github.com/lfcleo/fc-admin/blob/main/src/store/system/global.ts#L27)

## Add a quick theme color list

The `colorList` field in the `src/layout/views/settings.vue` file can modify the shortcut theme color.

[show code](https://github.com/lfcleo/fc-admin/blob/main/src/layout/views/setting.vue#L104)

# Dark Mode

The `sysDark` field in the `src/store/system/global.ts` file sets the default bright/dark mode.

[show code](https://github.com/lfcleo/fc-admin/blob/main/src/store/system/global.ts#L29)

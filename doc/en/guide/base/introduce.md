# Introduction

FC-Admin is an open-source, free, and ready-to-use template for admin panel systems. It is developed with the latest mainstream technologies including Vue3, Vite, Element-Plus, TypeScript, Pinia, and Tailwindcss. It helps you accelerate development speed and save a lot of time and cost.

## Preview

https://www.fcadmin.fun

## Complete Version

GitHub：https://github.com/lfcleo/fc-admin

## Thin Version

The streamlined version, derived from the FC-Admin framework, is better suited for straightforward development. Developers can focus on enhancing this project rather than subtracting features from the full version.

GitHub：https://github.com/lfcleo/fc-admin-thin

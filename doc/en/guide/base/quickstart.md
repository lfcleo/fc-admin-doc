# Quickstart

## Prerequisites

Please install `node`和`npm`。

Author's version

node：`v20.13.1`

npm：`v10.5.2`

## Pull Code

Complete Version

```shell
git clone https://github.com/lfcleo/fc-admin.git
```

## install

Enter the project folder using the terminal

```shell
npm install
```

## Running

```shell
npm run dev
```

Enter in browser` http://localhost:2800 `
